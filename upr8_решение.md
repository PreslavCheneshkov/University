### Отговори на въпросите:

**1. Възможно ли е от вашата рисунка да се види как работи интерфейса?**

Да, частично. От скицата се вижда основната структура на интерфейса:
- Менюта в горната част за достъп до функции
- Лента с инструменти (toolbar) с бързи бутони
- Лента с линийка за форматиране
- Основна работна площ за текст
- Статус бар в долната част

Обаче е трудно да се представи динамиката - как се отварят менюта, как текстът се форматира в реално време, как работят drop-down менютата и т.н.

**2. Какви проблеми срещнахте?**

- **Представяне на динамиката:** Много трудно е да се покаже как падащите менюта се разгръщат, как се променя форматирането в реално време
- **Идейният модел:** Трудно е да се представи WYSIWYG концепцията (What You See Is What You Get) само със статична скица
- **Детайлите:** Невъзможно е да се начертаят всички икони и опции за 5 минути
- **Интерактивността:** Не може да се покаже как курсорът се движи, как се селектира текст, как работи scroll-ването
- **Контекстни менюта:** Не се вижда функционалността при десен клик

**3. Какви графични техники използвахте?**

- **ASCII символи:** За създаване на прости рамки и разделители
- **Емоджита/символи:** За представяне на икони (📁, 💾, 🖨️)
- **Букви в квадратчета:** За бутони ([B], [I], [U])
- **Линии и рамки:** За разграничаване на различните области
- **Текстови етикети:** За именуване на менюта
- **Опростена перспектива:** 2D изглед отгоре
- **Символна нотация:** (|) за курсор, (▼) за падащо меню

## Задача 2: Разработване на графичен речник

### Избран модел: "Човек-звезда" (Star-person)

**Основен модел:**
```
    O      (глава)
   /|\     (тяло и ръце)
   / \     (крака)
```

### 5 разновидности на модела:

**1. Голям човек:**
```
      O
     /|\
    / | \
     / \
    /   \
```

**2. Малък човек:**
```
  O
 /|\
 / \
```

**3. Човек гледащ наляво:**
```
   O<
  /|\
  / \
```

**4. Седнал човек:**
```
   O
  /|\
  / \___
 /      \
```

**5. Стар човек (с бастун):**
```
    O
   /|\
   / \ |
      |
```

**6. Човек, който работи на компютър:**
```
    O
   /|\
   / \ 
  [==] ← компютър
```

**7. Бягащ човек:**
```
   O
  -|\
  / >
```

### Допълнителни елементи на графичния речник:

**Компютърна система:**
```
┌────────┐
│ [====] │  (екран)
│        │
└───┬┬───┘
   ████     (клавиатура)
```

**Сървър:**
```
┌─────────┐
├─────────┤ (стек от сървъри)
├─────────┤
└─────────┘
```

**Мрежа/комуникация:**
```
 [PC]──────[PC]
   │         │
   └────[HUB]────[PC]
```

**Стрелки за поток:**
```
→  ←  ↑  ↓  ⇄  ⇅  ↻  
```

## Задача 4: Визуални метафори за програмни понятия

### EDIT (Редактиране)
```
Словесна метафора: "Поправка", "Корекция"

Визуална метафора:
   ┌─────┐      ┌─────┐
   │Text │  ✎   │TEXT │  (молив променя текст)
   └─────┘  →   └─────┘

Алтернатива:
   ABC  →  ✂  →  ✎  →  A✓C  (рязане и поправка)
```

### DEBUG (Дебъгване)
```
Словесна метафора: "Лов на бъгове", "Почистване"

Визуална метафора:
   ┌────────┐           ┌────────┐
   │ 🐛 🐛  │    🔍     │  ✓✓   │  (намиране и премахване на бъгове)
   │ CODE  │    →      │ CODE  │
   └────────┘           └────────┘

Алтернатива:
    [⚠]  →  [🔍]  →  [🔨]  →  [✓]
```

### STRING (Низ)
```
Словесна метафора: "Нишка", "Верига от символи"

Визуална метафора:
   ╭─A─B─C─D─╮
   │         │  (символи нанизани на нишка)
   ╰─────────╯

Алтернатива:
   "H-e-l-l-o"  (символи свързани с тирета)
   
   или: 📿 (мъниста на конец)
```

### EXECUTE (Изпълнение)
```
Словесна метафора: "Стартиране", "Действие"

Визуална метафора:
   [▶] CODE  →  ⚙⚙⚙  →  ✓  (старт, процес, завършване)

Алтернатива:
        🏃
   ─────────────────▶  (бягане към цел)
        GO!
```

### DECLARE (Деклариране)
```
Словесна метафора: "Обявяване", "Етикетиране"

Визуална метафора:
   ┌───┐
   │ ? │  →  📛NAME  →  ┌───────┐
   └───┘              │ NAME  │  (поставяне на етикет)
                       └───────┘

Алтернатива:
   [ ]  →  "X"  →  [X]  (именуване на празна кутия)
```

### LOOP (Цикъл)
```
Словесна метафора: "Завъртане", "Повторение"

Визуална метафора:
      ↻
   ┌──────┐
   │ ↺    │  (кръгово движение)
   └──────┘

Алтернатива:
   START → [1] → [2] → [3] → ↺ START
   
   или: ∞ (безкрайност)
```

### FIELD (Поле)
```
Словесна метафора: "Място", "Контейнер"

Визуална метафора:
   ┌──────────────┐
   │ NAME: [____] │  (място за данни)
   └──────────────┘

Алтернатива:
   [  📦  ]  (кутийка за съхранение)
   └Label ┘
```
